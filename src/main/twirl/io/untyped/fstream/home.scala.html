@(msg: String)

Welcome to Scalaz-Stream Demo Application built with @msg!

The service is running with HTTP4S Blaze Server and connecting it with the Twitter Stream Source. We are using Websocket from Http4s as Sink.

The code is shamelessly copied from Http4S Demo application available on github.
Both http4s and blaze are works in progress: if you find bugs or have suggestions, please submit an issue.
